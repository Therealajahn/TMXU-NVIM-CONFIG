# TMXU-NVIM-CONFIG
My personal Tmux and NVIM config files

# TMUX usage
### Enter Copy Mode:
    Ctrl+A+[
### Cancel Copy Mode:
    Ctrl+c
### Highlight selection:
    Space
### Confirm Selection 
    Enter   
### Cancel selection
    Escape
### Paste
    Ctrl+A+]
   
   
# NVIM Usage 
## Start Bracy (live server)
    :Bracey
## Special Bindings
  ### Insert Mode
     Space
  ### Normal Mode
     (fast) JK
